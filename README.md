# Finances Control System

A Python CLI program to manage personal expenses efficiently.

## Features
* **Create:** Add new expenses easily.
* **Read:** View your full list of expenses.
* **Search:** Find specific records by name.
* **Delete:** Remove unwanted entries.
* **Persistence:** Automatically saves data to a JSON file.

## Technologies
* Python 3
* JSON
* Git & GitHub
